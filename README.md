
# Marketing Campaign A/B/n Testing Analysis

## Business Problem

A fast-food company tested three different marketing promotions to identify which campaign generated the highest sales performance for a new product launch.

## Objective

The objective of this project is to determine whether statistically significant differences exist among the promotion groups using ANOVA and Tukey HSD testing.

## Tools & Libraries

- Python
- Pandas
- Seaborn
- Matplotlib
- SciPy
- Statsmodels

## Analysis Performed

- Exploratory Data Analysis (EDA)
- Descriptive Statistics
- Boxplot Visualization
- Confidence Interval Analysis
- ANOVA Testing
- Tukey HSD Post-hoc Analysis

## Key Findings

- Promotion 2 performed significantly worse than Promotions 1 and 3.
- No statistically significant difference was found between Promotions 1 and 3.

- A/B/n Testing
- Statistical Hypothesis Testing
- ANOVA
